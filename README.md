# Yemekhane Kayıt Sistemi (C Dili)

Bu proje, **C dili** kullanılarak geliştirilmiş basit bir **yemekhane kayıt sistemi** uygulamasıdır.  
Öğrencilerin aylık yemek kayıtlarını yapabilmesini, iptal edebilmesini ve mevcut kayıtları listeleyebilmesini sağlar.  
Kayıtlar dosya üzerinde saklanarak program kapatılsa bile korunur.

---

## Özellikler

 -Öğrenci bazlı aylık kayıt sistemi   
 -Kayıt ekleme                      
 -Kayıt iptal etme                          
 -Mevcut kayıtları listeleme           
 -Dosyaya kayıt (kalıcı veri saklama)          
 -İç içe döngü ve matris kullanımı       
-`kayitlar.txt` dosyası ile okuma/yazma işlemleri      

---

## Kullanılan Teknolojiler

- **Programlama Dili:** C
